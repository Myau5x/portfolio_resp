My Role/Challenges

I used API from openweathermap.org, and it provides forecast based on latitude and longitude and not by city name. So to provide user forecast by city name I first sent request for current weather and it gives me coordinates of city, than I sent another API request for forecast and parse responce to show it user friendly way.
